Spring Boot ile tasarlanan bir e-ticaret sitesi backend servisi. Hesap, ürün oluşturma, sepete ekleme, sipariş verme ve ödeme yapma hizmetlerini sunan event-driven bir hizmet.

An e-commerce website backend service designed with Spring Boot. It's an event-driven service offering account creation, product creation, cart management, order placement, and payment processing.
